# Linux Log Monitoring Toolkit

A lightweight monitoring tool built with Bash and Python to analyze and track Linux system logs.

## Features
- Log parsing and keyword detection
- Real-time monitoring (tail-like behavior)
- Email alerts via SMTP
- Configurable via YAML
- Dockerized deployment

## Tech Stack
- Python
- Bash
- Linux
- Docker

## Usage

### Run Python version
```bash
cd python
python monitor.py
